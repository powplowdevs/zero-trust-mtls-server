Enroll New Device

Register this device for certificate-based access. An admin must approve your request before access is granted.

1

Generate your key & request

Run this on the device you want to enroll. Your private key (client.key) is created locally and never leaves your machine. Replace YOUR_USERNAME with the exact username you'll enter below — the certificate's name must match or approval will be rejected. 

openssl genrsa -out client.key 4096
openssl req -new -key client.key -out client.csr -subj "/CN=YOUR_USERNAME"

This produces client.csr. Open it and paste its full contents (including the BEGIN/END lines) into the form below. 

cat client.csr
2

Submit your request

Must match the /CN= value in your command above.

Your signed certificate will be emailed here once approved.

3

After approval — install your certificate

(do this once you get the email)

Once an admin approves your request, you'll receive a signed certificate by email. Save it next to your client.key as client.crt, then bundle the two into a single .p12 file your browser can import.

a) Save the emailed certificate as client.crt, then verify it:

openssl x509 -in client.crt -noout -subject

Should print your username as the subject CN.

b) Bundle cert + key into a .p12 (you'll set an export password):

openssl pkcs12 -export -inkey client.key -in client.crt -out client.p12 -name "pi-ZTNA"

If it errors with a key/cert mismatch, the cert wasn't issued from this key's request — re-check you used the right files.

c) Import client.p12 into your browser:

Firefox

Settings → Privacy & Security → Certificates → View Certificates → Your Certificates → Import → choose client.p12 → enter the export password.

Chrome / Edge

Settings → Privacy and security → Security → Manage certificates → Your certificates → Import → choose client.p12. Chrome uses your OS certificate store (Keychain on macOS, the system store on Windows, NSS on Linux).

After importing, open the login page — your browser will prompt you to present the certificate. Pick the pi-ZTNA one, then sign in with your username and password.
